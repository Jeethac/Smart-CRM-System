## 🎯 2. Objective

The objective of this project is to provide a centralized system where businesses can store customer information, manage leads, track follow-ups, and monitor staff performance, reducing manual effort and improving productivity.

---

## ⚙️ 3. Working of the System

The system works based on user roles:

* The **Admin** logs into the system and manages staff users
* **Staff members** log in using their credentials
* Staff can **add and manage customer leads**
* System will show Follow-up reminders
* The system can calculate lead scores automatically
* All data is stored securely in the database
* The Admin can monitor all activities and system performance

---

## 🏢 4. Business Use

This system can be used in:

* Sales and marketing companies
* Service-based organizations
* Small and medium businesses

It helps in:

* Managing customer records
* Tracking follow-ups with clients
* Prioritizing leads using scoring
* Improving team coordination
* Increasing business efficiency

---

## 🚀 5. Key Features

* Secure Login & Authentication
* Admin Dashboard
* Lead & Customer Management
* Follow-up Reminder
* Auto Lead Score Calculator
* Staff/User Management
* Organized Data Storage

## ▶️ 7. How to Run the Project

Follow the steps below to run the project on your system:

### 🔹 Step 1: Open Project Folder

Open your terminal and navigate to the project directory:

```bash
cd smart_crm
```

---

### 🔹 Step 2: Go to Project Folder

```bash
cd crm_project
```

---

### 🔹 Step 3: Activate Virtual Environment

```bash
venv\Scripts\activate
```

---

### 🔹 Step 4: Run the Server

```bash
py manage.py runserver
```

---

### 🔹 Step 5: Open in Browser

```text
http://127.0.0.1:8000/
```

---
